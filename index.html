Welcome to FitLog

Set up Ollama for free AI coaching — one time, 5 minutes

1
Install Ollama
Go to ollama.com → Download for Mac/Windows/Linux → Install
2
Pull the AI model
Open Terminal and run:
ollama pull gemma3:4b
~2.5 GB download. gemma3:4b = fast, quality reviews.
3
Start Ollama
ollama serve
Keep this running when using AI Review on laptop.
4
Log your day → click AI Review
No internet, no API key, no cost. Ever.
💡 No laptop? Use Copy Prompt → paste into ChatGPT.com or Claude.ai (both free). Works on mobile.
